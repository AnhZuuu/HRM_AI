## ğŸš€ Quick Start

### Prerequisites
- Node.js (version >=16.0.0)
- npm or yarn (Recommended: pnpm)


### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AnhZuuu/HRM_AI.git
   cd HRM_AI
   ```

2. **Install dependencies:**
   ```bash
   pnpm install
   ```

